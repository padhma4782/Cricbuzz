# Cricbuzz LiveStats - Cricket Analytics 

This project is for a Cricket Analytics Dashboard using:
- Python (Streamlit)
- MySQL
- REST API (requests) 


## Quick start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure MySQL db 
4. Create credentials for crickbuz app.
5. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## Project structure
See files under `pages/` and `utils/`. Fill API keys for live data.
